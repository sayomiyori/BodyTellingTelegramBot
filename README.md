# Telegram-бот «Тело говорит»

Клуб бережных тренировок — бот по ТЗ: онбординг, ежедневный опрос, подбор тренировок, огоньки, заморозки, достижения, прогресс, напоминания, вопрос тренеру.

## Установка

```bash
cd "d:\Programming\telegram bot"
pip install -r requirements.txt
```

## Настройка

1. Создай бота в Telegram через [@BotFather](https://t.me/BotFather), получи токен.
2. Скопируй `env.example` в `.env` и заполни:

```env
TELEGRAM_BOT_TOKEN=123456:ABC...
CHAT_LINK=https://t.me/your_club_chat
TRAINER_CHAT_ID=123456789
ADMIN_IDS=123456789
```

- `CHAT_LINK` — ссылка на закрытый чат клуба.
- `TRAINER_CHAT_ID` — ID чата или лички тренера для пересылки вопросов (узнать ID можно через @userinfobot).
- `ADMIN_IDS` — твой Telegram user id (через запятую, если несколько).

3. Время напоминаний и «конца дня» можно поменять в `config.py`:
   - `REMINDER_TIMES` — часы для утро/день/вечер (по умолчанию 8, 13, 19). Учтено время UTC: для Москвы (UTC+3) ставь 5, 10, 16.
   - `END_OF_DAY_HOUR` — час проверки «не тренировалась» и предложения заморозки (по умолчанию 22, UTC).

## Запуск

```bash
python bot.py
```

Или с загрузкой переменных из `.env` (установи `python-dotenv` и добавь в начало `bot.py` загрузку `.env`).

## Структура проекта

- `bot.py` — точка входа, обработчики, джобы напоминаний.
- `config.py` — токен, ссылки, ID, пути.
- `database.py` — SQLite: пользователи, опросы, вопросы тренеру.
- `messages.py` — все тексты бота (можно править без кода).
- `keyboards.py` — кнопки и клавиатуры.
- `workouts.py` — подбор тренировки по ответам опроса.
- `workouts.json` — библиотека тренировок (добавляй ссылки на видео и теги).
- `ИНСТРУКЦИЯ_ПО_ТЗ.md` — подробная инструкция по ТЗ.

## Библиотека тренировок

Файл `workouts.json`: каждая тренировка с полями:
- `id`, `title`, `duration`, `type` (силовая, растяжка, дыхание, экспресс),
- `energy_level`, `cycle_phase`, `time_slot`, `goal` — массивы тегов для подбора,
- `equipment`, `link` — что нужно и ссылка на видео/аудио.

Подставь реальные ссылки вместо `https://example.com/...`.

## Команды

- `/start` — приветствие или главное меню.
- `/stop` — отключить рассылку и напоминания.

## Чек-лист по ТЗ

См. файл **ИНСТРУКЦИЯ_ПО_ТЗ.md** — там полный разбор пунктов ТЗ и что проверить перед запуском.
