# Полный тест функционала бота «Тело говорит»

## Автоматические тесты

Запуск:
```bash
cd "D:\Programming\telegram bot"
python -m pytest tests/ -v
```

Покрыто:
- **База данных:** создание пользователя, обновление, состояние, серия, заморозки, опросы, вопросы тренеру, напоминания, сброс месяца.
- **Тренировки:** загрузка из JSON, подбор по ответам, категории по типу.
- **Обработчики:** /start (новый и после онбординга), /stop, онбординг (познакомимся, возраст, цели, ввод имени), главное меню (тренировка на сегодня, мой прогресс), опрос (4 шага + выдача тренировки), реакция после тренировки (Супер → запись достижения).

## Ручная проверка в Telegram

Проведи сценарии ниже в реальном боте (запущен `python bot.py`).

### 1. Онбординг (новый пользователь)

| Шаг | Действие | Ожидание |
|-----|----------|----------|
| 1.1 | Отправить `/start` | Приветствие + кнопки «Познакомимся» / «Сразу в клуб» |
| 1.2 | Нажать «Познакомимся» | Текст про ценности, затем вопрос «Как к тебе обращаться?» |
| 1.3 | Написать имя (например Маша) | Вопрос «Сколько тебе лет?» + кнопки возрастов |
| 1.4 | Выбрать возраст (например 25-30) | Вопрос про запрос + кнопки целей (несколько выборов) |
| 1.5 | Выбрать 1–2 цели, нажать «Готово» | Текст «Супер, Маша! Ты в клубе» + кнопки «Перейти в чат» / «На главную» |
| 1.6 | Нажать «На главную» | Появляется главное меню (кнопки) |

**Альтернатива:** «Сразу в клуб» → сразу вопрос имени (шаг 1.3).

### 2. Главное меню

| Шаг | Действие | Ожидание |
|-----|----------|----------|
| 2.1 | Нажать «Тренировка на сегодня» | Первый вопрос опроса (состояние и энергия) |
| 2.2 | Нажать «Мой прогресс» | Блок с серией, рекордом, месяцем, заморозками, достижениями |
| 2.3 | Нажать «Чат клуба» | Сообщение со ссылкой из .env |
| 2.4 | Нажать «Напоминания» | Кнопки: Утро / День / Вечер / Не надо |
| 2.5 | Выбрать «Утро» | Подтверждение «Буду напоминать…» |
| 2.6 | Нажать «Мои заморозки» | Текст «Доступно X из 3», при X>0 — кнопка «Потратить заморозку сейчас» |
| 2.7 | Нажать «Вопрос тренеру» | Вопрос «Напиши свой вопрос…» |
| 2.8 | Написать текст вопроса и отправить | «Твой вопрос передан тренеру» + главное меню |
| 2.9 | Нажать «Библиотека тренировок» | Кнопки категорий (Силовые, Растяжка и т.д.) |
| 2.10 | Выбрать категорию | Список тренировок (название + минуты) |

### 3. Опрос и выдача тренировки

| Шаг | Действие | Ожидание |
|-----|----------|----------|
| 3.1 | «Тренировка на сегодня» → выбрать энергию (например Бодрячком) | Вопрос про фазу цикла |
| 3.2 | Выбрать фазу (например Овуляция) | Вопрос «Сколько времени есть?» |
| 3.3 | Выбрать время (например 15-20 мин) | Вопрос «Какая цель на сегодня?» |
| 3.4 | Выбрать цель (например Сила) | Сообщение с названием тренировки, временем, инвентарём + кнопка «Начать тренировку»; ниже — «Как прошла тренировка?» с реакциями |
| 3.5 | Нажать «Супер» или «Молодец» | Поздравление с серией (огонёк), при первой тренировке — достижение «Первый шаг» |
| 3.6 | (опционально) Выбрать «Слилась» | Вопрос «Что помешало?»; после ответа текстом — «Записала…» и меню |

### 4. Мотивация

| Шаг | Действие | Ожидание |
|-----|----------|----------|
| 4.1 | После 1-й выполненной тренировки открыть «Мой прогресс» | Серия 1 день, достижение «Первый шаг» |
| 4.2 | В разделе «Мои заморозки» нажать «Потратить заморозку сейчас» (если есть) | «Заморозка потрачена», счётчик «X из 3» уменьшился |

### 5. Команды

| Шаг | Действие | Ожидание |
|-----|----------|----------|
| 5.1 | Отправить `/stop` | «Рассылку отключила…» |
| 5.2 | Снова отправить `/start` | Снова главное меню (онбординг не повторяется) |

Если что-то ведёт себя иначе — опиши шаг и ожидание, можно будет поправить логику или тесты.
